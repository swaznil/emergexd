# emergexd

Simple particle simulators, using rules of repulsion and attraction to in real time, creating complex emergent patterns and behaviors from simple rules.

---

## For Life01 (First attempt)

#### Features

- Real time particle simulation
- Pan camera controls + Zoom
- Bult-in presets
- Mutiple particle group supported
- Adjustable attraction/repulsion
- Spatial grid optimization for better performance

#### Controls

| Control | Action |
|---|---|
| Right click + drag | Pan |
| Mouse wheel | Zoom |
| Positive values | Attraction |
| Negative values | Repulsion |
| Groups | Create from the left panel or pick presets |

#### Tech Used

- HTML 5 with canvas
- JavaScript
- CSS

PixiJS rendering library was used
